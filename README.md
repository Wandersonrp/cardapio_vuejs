# Cardápio Digital

Projeto Cardápio Digital de curso de VueJs 3.

<br>

### Tecnologias utilizadas 

* VueJs 3;
* Json-Server;
* Axios.
  
<br>

### Instalação 
```
npm install
```

### Rodar o Json-Server dentro da pasta /src
```
npx json-server --watch data/db.json
```

### Rodar o VueJs 3
```
npm run serve
```
